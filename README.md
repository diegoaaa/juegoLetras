#Juego de Letras

Juego de letras es un Juego para poner a prueba la memoria a corto plazo

## Reglas del juego

- El juego inicia alumbrando una tecla en pantalla la cuál el usuario deberá presionar en su teclado

- A medida que se vaya presionando la tecla correcta se iniciará un nuevo nivel con una tecla mas para presionar

- El juego consta de 15 Niveles dónde al finalizar las 15 teclas presionadas correctamente mostrará que el usuario ha ganado y preguntará si desea jugar de nuevo

- Si en algún momento el usuario presiona una tecla que no sea de las que se mostraron en pantalla o en el orden inadecuado, el sistema le mostrará un mensaje informándole que ha perdido y si desea iniciar una nueva partida

## Instalación

```
npm install juegoLetras
```

## Uso

-Haz clic en el archivo juego.html y espero que te guste
